# AnalyticsPro — Business Analytics Dashboard

🔗 [Live Demo](https://analytics-dashboard-qdtm.vercel.app)

A business analytics dashboard built with Next.js, Supabase, and Recharts.

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Recharts
- Supabase

## Features

- Revenue, traffic and sales charts
- Transactions table with search and filter
- Date range filter
- Dark and light mode
- Real data from Supabase

## Getting Started

1. Clone the repo and install dependencies
   npm install

2. Add your Supabase keys to .env.local
   NEXT_PUBLIC_SUPABASE_URL=your_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key

3. Run the app
   npm run dev
